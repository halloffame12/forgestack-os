## Getting Started

### Prerequisites

- Node.js 18+ (or Bun if using Bun backend)





### Installation

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Update .env with your configuration
```

### Database Setup

```bash
# SQLite is file-based, no setup needed
# Database file will be created automatically
cd backend
npx prisma migrate dev
```

### Development

```bash
# Run both frontend and backend
npm run dev

# Or run separately
npm run dev:frontend
npm run dev:backend
```

## Authentication

This project uses built-in JWT authentication. Users can register and login through the API endpoints:

- `POST /api/auth/register` - Create a new account
- `POST /api/auth/login` - Login and receive JWT token
- Include the token in requests: `Authorization: Bearer <token>`
